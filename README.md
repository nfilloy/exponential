### Main Achievements

- **CESGA — AI Engineer Intern (Feb 2026–Apr 2026).** Worked on applied AI, supercomputing, RAG workflows, automation, and full-stack development in a real HPC environment supported by FinisTerrae III infrastructure.
- **FRAGATA — Hybrid RAG System for HPC Support Tickets (2026).** Co-authored and built a semantic retrieval system over 20+ years of CESGA Request Tracker history. The system combines multilingual embeddings, BM25, weighted reciprocal rank fusion, cross-encoder reranking, incremental ingestion, hot-swap reloads, and offloading to FinisTerrae III. Published on [arXiv](https://arxiv.org/abs/2604.13721).
- **Monte Carlo Cathedral — 2nd Place at Project Europe BCN (Apr 2026).** Built an AI investment agent for the Cala track that allocated $1M across 15+ NASDAQ stocks. The system used an LLM to generate causal hypotheses from a 2,000-entity / 3,000-relationship knowledge graph, propagated impact with BFS, ran 50,000 Monte Carlo scenarios, and optimized portfolio weights with CVaR. Best run: +66.4%; average across six configurations: +37.8%. [Read the write-up](https://alexlopezg.substack.com/p/we-finished-2nd-at-project-europe).
- **SHIELD — Phishing and Smishing Detection with NLP and XAI (Dec 2025).** Built an end-to-end cybersecurity system for analyzing SMS and emails with ETL pipelines, TF-IDF/n-gram extraction, URL and structural features, multi-model inference, and LIME/SHAP explainability. Best SMS model reached 0.9901 accuracy and 0.9965 ROC-AUC.
- **Computer Engineering at Universidade de Vigo (2022–2026).** Focused on software development, systems, databases, applied artificial intelligence, cybersecurity, and practical product engineering.
